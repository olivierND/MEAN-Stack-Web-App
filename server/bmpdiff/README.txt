L'éxécutable doit être roulé sur Fedora/L-4712. 
La commande à utiliser pour tester l'exécutable doit être fait à partir du dossier ./server.
L'images générée se retrouve dans le dossier ./server/<sortie>.

$ ./bmpdiff/bmpdiff [-partial] <original> <modifié> <sortie>

RUN COMMAND TO CREATE EXECUTABLE IN ./bmpdiff
make
